# Dentics - BackOffice

Gestion des rendez-vous, utilisateurs et messages côté administration.

## Structure
- `dashboard.php` : Tableau de bord
- `rendez_vous.php` : Gestion des rendez-vous
- `utilisateurs.php` : Liste et gestion des utilisateurs
- `messages.php` : Messages envoyés via le formulaire de contact
- `disponibilites.php` : Créneaux horaires des dentistes

---

Projet développé dans le cadre du module Web & BDD.
